# `@wmeimob/rich-text`

> 美萌富文本组件

## 目录

```js
├── src                   
│   ├── components/richText     # web端富文本组件

```

## 使用

对于后台管理系统。我们更建议你直接使用封装组件**ProFormRichText**

```tsx
import { MMRichText } from '@wmeimob/rich-text'

const Component = () => {

  return (
    <MMRichText fileUpload={upload} />
  )
}
```
